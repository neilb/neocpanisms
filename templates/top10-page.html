Release a new CPAN dist each month. How many months can you keep that up for? For the super-keen, there is also a "once a week" contest; the weeks start on Sunday and end on Saturday. In the still-active chains, where an author has a green background that means (s)he has already done a release in the current month or week. Where two authors have the same chain length, the one with most new dists in that period wins. The code is on github, where you can also log any bugs or suggestions.
